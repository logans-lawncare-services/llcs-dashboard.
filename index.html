Welcome to LLCS Dashboard
Track jobs, expenses, and earnings for Logan’s Lawncare Services in one place.

Use the tabs above to move between features:

  • Expenses: submit and review expenses. Admin approves or declines.
  • Jobs: add jobs, payments, notes, and track progress.
  • Jobs Columned: Kanban view of jobs by status (planned → in progress → done → paid).
  • Charts: visual overview of earnings, expenses, and worker stats.
  • Admin: account management, money totals, and exports (Founder/Admin only).

Data is stored in your browser’s local storage on this LLCS GitHub site. Clearing browser data or using a different device will reset the dashboard for that browser.
