Welcome to LLCS Dashboard
Track jobs, expenses, and money for Logan’s Lawncare Services in one place.

Use the tabs above to move between features:

  • Expenses: submit and review expenses.
  • Jobs: add jobs, payments, and notes.
  • Jobs Columned: Kanban view by status.
  • Charts: visual stats for jobs and money.
  • Admin: accounts, money, and data export (admins only).

Data is stored in your browser’s local storage on this GitHub Pages site. Clearing browser data or using another device will reset it for that browser.
